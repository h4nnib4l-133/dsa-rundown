# DSA Revision for HackerRank

100 problems across 10 topics. Write each solution in Python and C.

## Start Here

1. Open `PROGRESS.md` -- that's your checklist
2. Go to a topic folder (e.g., `01-binary-search/`)
3. Read `milestone.md` -- understand the pattern and study the examples
4. Write your solution in the `.py` and `.c` template files
5. Test, debug, check the box in PROGRESS.md
6. Move to next problem

## Structure

```
01-binary-search/         Binary search patterns and variations
02-dynamic-programming/   1D DP, 2D DP, knapsack, string DP
03-strings/               Sliding window, frequency maps, two pointers
04-linked-lists/          Pointer manipulation, fast/slow, merge
05-graphs/                BFS, DFS, topological sort, Dijkstra
06-arrays-sorting/        Two pointers, prefix sums, intervals
07-stacks-queues/         Monotonic stack, balanced brackets, deque
08-trees/                 Traversals, BST properties, recursion
09-recursion-backtracking/ Subsets, permutations, constraint solving
10-greedy/                Interval scheduling, greedy choices
```

Each topic has a `milestone.md` with: pattern explanation, problems list, hints, and template files.

## How to Use

**Python:**
```bash
python 01-binary-search/01_search_sorted.py
```

**C:**
```bash
gcc -o solution 01-binary-search/01_search_sorted.c -Wall && ./solution
```
